# My Userscripts

## installation
1. Install a Userscript addon

> Browser        | Addon
> -------------- | -------------------------------------------------------------------------
> Firefox        | [FireMonkey](https://addons.mozilla.org/en-US/firefox/addon/firemonkey/)
> other browsers | [Tampermonkey](https://www.tampermonkey.net/)

2. Click on the install link of the script
